# Currency Converter (Python)

## Overview

A command-line based Currency Converter developed in Python.  
This application enables users to convert monetary values between 
multiple international currencies using predefined exchange rates.

## Key Features

- Multi-currency conversion support  
- Command Line Interface (CLI) interaction  
- Validation for unsupported currency codes  
- Formatted output with 2 decimal precision  
- Modular and readable code structure  

## Supported Currencies

USD, EUR, GBP, JPY, AUD, CAD, CHF, CNY, INR, PKR, SGD, SAR, AED, KRW, MYR, LIRA, KWD, EGP, BDT, LKR.
